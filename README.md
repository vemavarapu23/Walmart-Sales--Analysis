### Retail Data Analysis and Visualization using Microsoft Excel

This project demonstrates advanced data cleaning, analysis, and visualization skills applied to a year of retail transactional data.

## Key Objectives:
* **Data Preparation:** Clean and structure raw sales data, handling missing values and ensuring data integrity for accurate analysis.
* **Performance Summarization:** Calculate core metrics and financial summaries including total profit, quarterly sales totals, and regional performance comparisons.
* **Interactive Dashboard:** Utilize advanced **Pivot Tables, complex formulas (VLOOKUP/INDEX),** and conditional formatting in **MS Excel** to create a dynamic, user-friendly dashboard.
* **Trend Identification:** Provide insights on sales trends, product seasonality, and opportunities for optimizing inventory or focusing marketing efforts.

## Tools Used:
* Microsoft Excel (Data Cleaning, Formulas, Pivot Tables, Visualization)
